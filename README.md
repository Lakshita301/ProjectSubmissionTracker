# Project Submission Tracker

A simple web application to track student project submissions with database support. This repository contains a Python backend, HTML/CSS frontend templates, and database schema to manage students and their project submissions.

## Overview

This project helps instructors and administrators:

- Store and view student project submissions
- Track submission dates, project details, and student information
- Use an intuitive web interface to input and update records
